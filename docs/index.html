xsize (newSize)
对node:size函数的扩展 和size不同的是，size必须指定宽高，这个可以指定宽或者高

Parameters:

  • newSize 新的节点大小，table

Returns:

    cc.Node

Usage:

    node:xsize({width = 100, height = 200})

node:size(100, 200)
